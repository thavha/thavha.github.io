About

I started by learning Python as my first programming language on my own and obtained a certificate Titled: Microsoft Technology Associate (MTA): Introduction to Programming Using Python. I am interested in learning new technologies such as cloud computing and Data Engineering in the cloud. Due to my interest Data Engineering in the cloud I started learning Microsoft Azure, to test my skills and knowledge I set for 3 Microsoft Exams and passed all of them to get certified on Azure cloud fundamentals, Azure Data fundamentals and Azure AI fundamentals. I had an opportunity to work at Telkom as Data engineer intern where i practiced the skilled i have learned. I am now a Data and BI developer at BidvestBank where I am responsible of building ETL, storing and extracting data, and developing BI reports.
